14:00 16:00 Meu Primeiro Aplicativo Android Segundo a Worldwide Mobile Phone Tracker (IDC), mais de 80% dos dispositivos móveis utilizam o Google Android como Sistema Operacional. A Google Play (loja de aplicativos oficial) possui quase 2 milhões de aplicativos publicados. Isso implica afirmar que existem muitos aplicativos para muitos usuários. 14:00 16:00 Workshop de Arduino Nesse workshop veremos uma introdução à sistemas embarcados utilizando a placa Arduino, uma ferramenta bastante poderosa e, ao mesmo tempo, simples. Com ela é possível prover soluções eletrônicas que resolvem problemas do dia a dia. A lista de aplicações é praticamente infinita. Vai depender somente da sua criatividade!
14:00 16:00 Introdução às Redes de Computadores Utilizando meios de transmissão com e sem fio, a Internet conecta computadores de pequeno e grande porte, permitindo que usuários compartilhem uma enorme quantidade de informações. Para um melhor entendimento sobre seu funcionamento, é preciso algum conhecimento sobre as redes de computadores, pois a Internet trata-se de uma "rede de redes". Com a finalidade de apresentar alguns conceitos básicos sobre as redes de computadores, no minicurso serão abordados alguns temas como: classificação e dispositivos de rede, transmissão de dados e endereço IP. Também serão realizadas algumas práticas utilizando o Packet Tracer, que é um software desenvolvido pela Cisco para simulação e análise de redes.
14:00 16:00 Iniciando o Desenvolvimento de Jogos O setor de games no Brasil cresceu uma média de 12% nos últimos cinco anos. Os jogos digitais são considerados como atividade da Economia Criativa, pois aliam a criatividade ao seu valor econômico, e ultrapassaram o viés do entretenimento. Hoje, os games são importantes ferramentas pedagógicas, utilizadas em escolas e em treinamentos empresariais. Tendo isso em vista, como pontapé inicial, vamos estudar o desenvolvimento de jogos com Unity3D.
14:00 16:00 Python para Todas Aprenda a programar de uma maneira fácil e diverta. Python é uma linguagem de programação simples e poderosa. Considerada uma das melhores linguagens para programadores iniciantes. Nesse minicurso, você aprenderá o básico da linguagem e verá que grandes coisas são feitas com poucas linhas.
